# Mini Bug Tracker

A simple web application to track bugs and issues in small projects using browser local storage.

## Features

- Add bug title and description
- Set bug status (open or closed)
- Delete bugs
- Store bugs locally
- Simple and clean interface

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Use

1. Enter bug title and description
2. Select status
3. Click add bug
4. View and manage bugs

## Future Improvements

- Edit bug
- Filter by status
- Search bugs
- Add priority levels
